Did you notice the change in Trakzee Dashboard?

The advanced dashboard of Trakzee has new updates. This dashboard will enable customisation and also will simplify your work.

26 December, 2019 Latest Update

We understand, you always look for platforms that are easy to use. So, Trakzee platform always tries to be user-friendly. Today, we will discuss the new change in the Trakzee web platform.   

Trakzee introduces a brand new customisable Dashboard. It is easy to understand and also provides all the useful information you need in a single screen. It has all the data ranging from Vehicle Status and Utilization to Expense, Reminders, and even your Object Health.

Let’s see the new changes in the platform. The below image is exactly how new dashboard will look.

New features…

  • Fleet Distance and Duration Summary

The top of the dashboard shows the total amount of distance travelled by your fleet for the day along with the trip and duration related information such as a number of trips of your fleet, total driving duration, idle duration, stop duration, inactive duration and also the average and maximum speed attained by the vehicles of your fleet.

  • Object status and map

Next is the part shows you the number of objects in each status such as running, idle, stop, inactive and no data.

Then is the map, it will show the live location of each object in your fleet. When you click on the colour that represents each status, the map will show only those vehicles. When an object is clicked, then the details of that object, such as its name, current speed and the live address will be shown. You also have the option to toggle between Cluster view and Traffic details.

  • Graph

The screen provides various graphs which are:

  1. Fleet Utilization: This graph shows the running duration and distance by the objects in each company
  2. Fleet Idling: This graph shows the total idle duration for each company
  3. Expense History: This graph shows the expenses for fuel and maintenance are done for each company along with the total amount
  4. Project vs Total Object: This graph will show you the total number of vehicles in each project for the admin or reseller
  5. Fleet Parking: This graph shows the total stoppage duration for each company
  6. Object Health: This section will show you the number of objects with their health state which is minor, major and critical
  7. GPS Model Summary: This graph shows the number of GPS Device Models used by each company
  8. Object Statics: This graph shows the number of Objects added and deleted for the selected duration.
  9. Maintenance Reminder: This section shows you the number of Due, Overdue and Upcoming maintenance reminders
  10. Fleet Violation: This graph shows the number of violationdone by the Objects under each company.

Let’s see the options of the graph by taking the example of Fleet Idling.

To switch between various graphs you can click on the icon. Then you can select any graph you prefer as shown below:

Also, you can switch to a textual view by clicking on the icon and it will appear as below:

To open the graph for a different date you can select the required duration from the date picker:

The above 3 options will remain the same for each graph.

Now, you can go into the detailed graph by clicking on the colour represented by any company and it will show the running duration and distance of the objects in that company.

You can go into further details by clicking on the colour represented by any object. This will open the graph which shows the running duration and distance travelled by the selected object:

Similarly for each graph, you will get all the data required by selecting the required graph or the icons.

  • Renewal Reminders

This section will show you the information of the various renewals such as Driver’sLicense, Insurance etc. When you click on the numbers mentioned under Due, Overdue or Upcoming then it will show you the vehicle details for which there is a renewal. 

These were the various graphs of the new Dashboard which shows various useful information regarding your fleet. But there is an additional feature in the new Dashboard which is customization.

  • Customisation

You will be able to customise your dashboard for which you have to simply click on the icon from the top right corner and you can remove, rearrange and resize each every graph and section of your Dashboard.

All you have to is drag each section to particular places of the screen where you would like to view it. And to resize any section you can simply use the resize arrows to increase or decrease their size. your Dashboard will be shown as per your configuration on all your future logins to the portal.

Conclusion

The new dashboard will take you to a new world of simplified information. For information like this be connected to us.
